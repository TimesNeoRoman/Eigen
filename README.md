# 주사위 베팅 게임.

## 1. 프로젝트 개요.
목표: 

베팅을 통해 초기 코인을 최대한 증대시켜, 가장 높은 이익률을 달성한다.

규칙: 

3개의 주사위를 굴려, 합이 11을 초과하면 'Over' 승리, 11 이하면 'Under'가 승리한다. 어느 시점에 배팅했느냐에 따라 배당이 달라진다.

주사위를 굴리기 전 0개, 1개, 2개의 주사위가 공개된 상태에서 베팅할 수 있으며, 각각 4배,3배,2배의 배당을 가진다.

## 2. 주요 기능

- 단계별 전략적 베팅: 주사위를 하나씩 굴리며 베팅 타이밍을 조절할 수 있다. 일찍 베팅할수록 배당률이 높다.

- 확실성 기반 배당 시스템: 승리가 거의 확실한 상황(예: 이미 주사위 합이 높은 경우)에서는 배당률이 자동으로 조정된다.

- 명언 API 연동: 베팅에 실패했을 때 위로와 조언이 담긴 명언을 API를 통해 실시간으로 가져와 보여준다.

- 전적 기록 시스템: 게임 종료 시 수익률을 계산하여 상위 5개의 최고 기록을 저장하고 표시한다.

- 파산 시스템: 코인을 모두 잃으면 파산 화면이 출력되며 게임이 종료된다.


## 3. 상세 메커니즘

- 이익률 상위 5개 기록
    

    정해진 7라운드를 끝마치거나 보유 코인의 개수가 0이 되는 경우, 게임이 종료된다.

    게임 종료 시, (최종 코인 - 초기 코인) / 초기 코인 * 100 공식을 사용하여 최종 이익률을 계산하고 사용자에게 보여준다.

    우측 패널에는 플레이했던 게임의 최고 이익률 Top 5 기록이 표시된다. 이 기록은 이익률이 높은 순서대로 정렬되며, 사용자에게 성취감을 제공하고 다음 플레이를 유도하기 위해 생성했다.

- 필승 상황시 배당 조정

    앞서 2개의 주사위를 굴린 후 베팅을 실시할 경우, 필승 상황이 발생한다. 이때 배당을 2배로 유지하는 것은 불확실성에 비해 과한 보상이다.

    따라서, 다음과 같은 조치를 하였다.

    확실성 조건 충족 (1.4배):

        이미 공개된 2개 주사위의 합이 11 이상인 경우 (Over가 거의 확정).

        이미 공개된 2개 주사위의 합이 5 이하인 경우 (Under가 거의 확정).

    이 경우, 배율은 1.4배로 낮아진다.

    불확실성 조건 (2.0배):

        위의 확실성 조건에 해당하지 않아, 마지막 주사위 하나에 의해 Over/Under가 갈릴 수 있는 경우 (예: 합이 6에서 10 사이).

    이 경우, 배율은 2.0배가 적용된다.
